# Mural de Comunicação Interna e Inclusiva - Indústria Sustentável

Este projeto foi desenvolvido como parte do **Projeto Integrador da Jornada Liga STEAM - Proz Educação**. Trata-se de uma solução digital voltada para otimizar a comunicação interna nas indústrias, promovendo três pilares essenciais: **acessibilidade, inclusão e sustentabilidade**.

## Funcionalidades e Diferenciais (Incrementos)
- **Banco de Dados Local:** Utiliza a Web Storage API (`localStorage`) para persistência de dados. As mensagens publicadas ficam salvas no navegador e não são perdidas ao atualizar a página.
- **Categorização de Avisos:** Permite filtrar os comunicados entre "Aviso Geral", "Ideia Sustentável" e "Alerta de Segurança", organizando visualmente por cores dinâmicas.
- **Acessibilidade Digital:** Inclui um botão de **Alto Contraste** para garantir uma navegação inclusiva para pessoas com baixa visão.
- **Sustentabilidade:** Reduz a zero o uso de papel para murais físicos de avisos na fábrica.
- **Gerenciamento:** Opção de excluir mensagens diretamente do banco de dados local.

## Tecnologias Utilizadas
- HTML5 (Estrutura semântica)
- CSS3 (Layout Responsivo e Modo de Alto Contraste)
- JavaScript (Manipulação de DOM e Persistência com `localStorage`)
